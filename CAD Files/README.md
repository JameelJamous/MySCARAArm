# CAD Files

The File nomenclature is as follows 

`SRM##-name` - SCARA ROBOT MODEL item no - name of what the model is

SRM Parts:
| Part No. | Description |
| ----------- | ----------- |
| 01 | Base of the arm |
| 02 | Joint |
| 03 | Lead Screw Rod |
| 04 | The Arm base |
| 05 | The Manipulator Base |
| 06 | The Pinion |

SRM Assembly:
| Assembly No. | Description |
| ----------- | ----------- |
| 01 | Main |
| 02 | Manipulator |
